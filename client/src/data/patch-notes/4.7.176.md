# Star Citizen 4.7.176 Patch Notes (DataForge Changes)

## New Vehicles Added
- **MISC Hull B** — Medium freight ship, 512 SCU cargo capacity (16 external cargo grids), 2-crew. A dedicated cargo hauler joining the Hull series lineup
- **Greycat UTV** — 2-seat utility buggy for ground logistics. Light cargo (2 SCU), multiple paint options

## Ship Updates
- RSI Hermes — AI civilian and criminal NPC variants added
- RSI Aurora GS SE — Minor modifications
- Drake Buccaneer — New red/black paint added
- MISC Raft — New white/blue paint added

## Contract Changes
- Foxwell Enforcement missions reworked — patrol, ambush, escort, ship wave attack, and destroy items missions all modified
- Courier (FTL) missions updated
- PvP bounty, ship heist, defend, and outpost infiltration contracts tweaked
- Tutorial Part 2 updated
- New dev test contract added (inactive — not player-facing)

## Crafting
- 2 Outlaw Legacy heavy armor helmet blueprints modified

## Harvestable / Mining
- ASD Delving asteroid RCD configurations adjusted (central, left, right positions)

## No Changes To
Mining stats, ore values, refinery methods, loot tables, weapon/ammo balance, reputation tiers, law system, factions, or starmap locations
